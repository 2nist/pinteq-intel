Five-stage intelligence pipeline
Stage
01
Intake & chain of custody
Files copied to drop folder. Intake form logged with case ID, date, source, and media type. Every file receives a unique intake record before any processing begins. Nothing enters the system without a logged receipt.
Intake form Case association Date stamp Source logging Chain of custody record
Smokeball API
/matters — link to case /matters/{id}/memos — log receipt
↓
Stage
02
Classification & normalization
Pipeline detects file type and assigns document category. Every file renamed to the agreed convention: [CASE_ID]_[DATE]_[SOURCE]_[TYPE]_[SEQ]. Original filename preserved in the index. Moved to structured folder hierarchy.
Type detection Category assignment Rename convention Folder placement Master index entry
Smokeball API
/files — upload normalized file /folders — place in structure /matters/{id}/tags — apply tags
↓
Stage
03
Content extraction
PDFs → text extracted via pdfplumber. CSVs → structured database rows. Audio/video → Whisper API transcription queue. Spanish-language material flagged for translation pass. All extracted content stored alongside the file record and made fully searchable.
PDF text extraction CSV parsing Whisper transcription Spanish translation Full-text search index
Smokeball API
/files — upload transcript PDF /matters/{id}/memos — key quotes
↓
Stage
04
Intelligence construction
Relationship graph built from phone records, contact lists, and social data. Case chronology reconstructed from events, file history, and communications. Witness statements cross-referenced for contradictions. Prosecution timeline gaps identified.
Relationship mapping Network graph Chronology engine Contradiction detection Witness tagging
Smokeball API
/contacts — add discovered people /contacts/{id}/tags — tag roles /events — populate timeline
↓
Stage
05
Intelligence delivery
Structured analytical products delivered directly into Smokeball as matter memos. B4 Witness Intelligence Briefs. Case intelligence summaries. Relationship maps. Chuck opens Smokeball and finds finished intelligence ready for trial preparation — no hunting required.
B4 witness briefs Subject profiles Case summaries Network visualizations Trial prep documents
